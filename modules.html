Available modules
Mobilid has a set of modules to improve the efficiency of your daily logistic and commercial operations.

Standard modules
The modules included in the base package you buy on Dolistore

Inventory module.

Module to do products inventory.

Different options to find the product:

  • Select from list. Searching, filtering and sorting possible.
  • Scan product by built-in camera barcode scanner.
  • Barcode scanning with optional Wireless bluetooth scanner.

Inventory capabilities:

  • Easy entry of product count.
  • Possibility to transfer to another warehouse.
  • Add barcode using scanner.
  • Add product photo using camera.
  • Show detailed product information.
  • Update lot data.
  • Add new lot.
  • Update sell price.

Order Picking module

Module to collect customer orders.

Simple order selection:

  • Selection from sorted and classified list.
  • Multi user, selected orders will be locked for other users.
  • Order code barcode scanning by built-in camera.
  • Order code barcode scanning via optional Wireless bluetooth scanner.

Picking capabilities:

  • Simple entry of picked product numbers.
  • Pick by scanning product barcode
  • Possibility of picking from multiple warehouses.
  • Show detailed order and product information.

Order module

Module to create customer orders.

Simple order creation:

  • Select customer from filterable, sortable and searchable list.
  • Add new customer. Can be company or private individual.
  • Use customer payment conditions.
  • Set shipment method.

Add products and services:

  • Add product or service from filterable, sortable and searchable list.
  • Add one product by scanning product barcode.
  • Add items to existing draft order.
  • Alter items in existing draft order.
  • Multi user, updating orders will be locked for other users.
  • Keep track of order price totals.
  • Update product sell price.
  • Grant total discount, will be distributed on all order items.

Commercial module

Module to contact Prospects and make appointments.

Easy contact your prospects and customers, make appointments using calendar.


Search prospects and contacts:

  • Select from searchable company list.
  • Filter company list on category, town and progress status.
  • Sort on all available parameters.
  • Processed company will be locked for other users.
  • Modify company attributes.
  • Add or modify contacts.
  • Call or send message to company or contact.

Agenda capabilities:

  • Browse and modify available agenda entry's.
  • Add new agenda task.
  • Search for free time-slots in agenda.
  • Task assignment to selectable user.

Add-On modules
Optional modules you can add to Mobilid. An activation key is required for each user. One key for each optional module is included when you buy the connector from Dolistore. Please contact for obtaining activation keys.

Dispatch module

Module to receive your supplier orders to your stock.


Simple order selection:

  • Selection from sorted and classified list.
  • Multi user, selected orders will be locked for other users.
  • Order code or Supplier code barcode scanning.
  • Product barcode scanning to get order(s) containing product.

Dispatch capabilities:

  • Overview of Ordered, already dispatched and Stock Qty.
  • Dispatch by scanning product barcode
  • Add missing product barcodes.
  • Add product photo using camera.
  • Compatible with lot/serial products, easy adding of product lot data.
  • Auto-fill lot, qty, sell before and expire dates from GS1 packages barcode.
  • Update product buy price.

Exit Stock module

Module for easy moving and removing items from your stock.


Different options to find the product:

  • Select from list, searching and sorting possible.
  • Scan product by built-in camera barcode scanner.
  • Barcode scanning with optional Wireless bluetooth scanner.

Stock exit capabilities:

  • Easy entry of exit count.
  • Possibility to change source stock location inside stock exit form.
  • Possibility to select a fixed destination stock location to move your stock to.
  • Add designation for stock exit, a list of used designations will be automaticly created for easy re-use.

Purchaser module

Module to create supplier orders.

Simple order creation:

  • Select supplier from filterable, sortable and searchable list.
  • Add or edit supplier.
  • Order by supplier, select product from supplier.
  • Order by product, add products to new or existing supplier order (replenishment).

Add products and services:

  • Add product or service from filterable, sortable and searchable list.
  • Add one product by scanning product barcode.
  • Add items to existing draft order.
  • Alter items in existing draft order.
  • Validate - Un-validate order.
  • Multi user, updating orders will be locked for other users.
  • Keep track of order price totals.
  • Update product buy price.
  • Grant total discount, will be distributed on all order items.
