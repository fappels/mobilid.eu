Mobilid
Mobilid est un solution client-serveur générique. Vous pouvez acheter une solution extensible, contenant une licence pour quelqeus modules standard. Extensible cela signifie on peut acheter des modules add-on et des modules fait sur-mesure. Pour activer ces modules, vous aurez besoin d'un code de licence de l'utilisateur. Ce code doit être entrée dans l'application cliente. Le client mobile peut être télécharger gratuit.

Mobilid Dolibarr ERP connector

Module de Dolibarr pour connecter Mobilid à la système ERP Open Source Dolibarr. Utiliser le système ERP Dolibarr pour faire la gestion administrative de votre client et du Dolibarr stock. Grâce à ce module développé par Z-Application, Mobilid peut vous aider de incrémenter la productivité de votre administration de votre entreprise.
Pour soutenir le projet Open Source Dolibarr, ce module peut être acheté sur le Dolibarr module Boutique.
