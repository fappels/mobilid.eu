Install
Go into menu Home - Setup - Modules - Deploy an external module and submit the module zip file.
Module 'SHIPMENT PACKAGES' is visible in list of modules (CRM Section) and can be activated and configured.

Features
Create final shipping packages to handover to transporter.
From a validated shipment you can create a package by selecting the items to put in package.

Please check also our other modules Mobilid, Block Outstanding and Backorder.
