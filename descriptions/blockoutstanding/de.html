Installieren:
Gehen Sie in das Menü Home - Einrichtung - Module - Stellen Sie ein externes Modul bereit und senden Sie die Zip-Datei des Moduls.
Modul 'Block Outstanding' ist in der Liste der Module sichtbar und kann aktiviert und konfiguriert werden.

Merkmale
Sperren Sie Vorschläge, Aufträge und Rechnungen, wenn der ausstehende Zahlungsbetrag das Limit überschreitet.
Sie können das Modul mit folgenden Einstellungen konfigurieren:
Das Modul verwendet das Limit, das in der Eigenschaft für ausstehende Kundenlimits festgelegt wurde. Wenn nicht festgelegt, können Sie einen Standardwert für alle Kunden definieren.
