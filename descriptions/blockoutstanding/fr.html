Installation
Allez dans le menu Accueil - Configuration - Modules - Déployer un module externe et soumettez le fichier zip du module.
Le module "Bloc Encours" est visible dans la liste des modules et peut être activé et configuré.

Fonctionnalités
Bloquez les propals, commandes, factures, commandes fournisseurs et factures fournisseurs si le montant du paiement en souffrance dépasse la limite fixée.

Le module utilisera la limite fixée dans la propriété de limite d'encours client, si elle n'est pas fixée, vous pouvez définir une valeur par défaut pour tous les clients.
