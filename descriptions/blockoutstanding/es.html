Instalar
Entra en el menú Inicio - Configuración - Módulos - Despliega un módulo externo y envía el archivo zip del módulo.
El módulo 'ExtDirect' está visible en la lista de módulos y puede ser activado y configurado.

Reportajes
Bloquear crear/modificar/validar las propuestas, pedidos, facturas, pedidos de proveedores y facturas de proveedores si la cantidad de pago pendiente excede el límite.
Bloque Crear en Entradas, intervenciones, contratos y proyectos.
Derecho del usuario a deshabilitar la edición del campo de límite pendiente de pago de terceros.
El usuario tiene derecho a que sólo se le avise y no se le bloquee.

El módulo utilizará el límite establecido en la propiedad de límite pendiente del cliente, si no está establecido se puede definir un valor por defecto para todos los clientes.
