Instalar
Entra en el menú Inicio - Configuración - Módulos - Despliega un módulo externo y envía el archivo zip del módulo.
El módulo 'ExtDirect' está visible en la lista de módulos y puede ser activado y configurado.

Reportajes
Bloquear crear/modificar/validar las propuestas, pedidos, facturas, pedidos de proveedores y facturas de proveedores si la cantidad de pago pendiente excede el límite.
Bloque Crear en Envío, Entradas, intervenciones, contratos y proyectos.
Bloquee la adición o actualización de líneas que provocarían que se superara el límite pendiente.
Mostrar el estado del bloque en la tarjeta de terceros y en la lista.
Derecho del usuario a deshabilitar la edición del campo de límite pendiente de pago de terceros.
El usuario tiene derecho a que sólo se le avise y no se le bloquee.
Si ha establecido el nivel de bloqueo en modo «Error», aún puede configurarlo para que solo se muestre una advertencia para su propio proveedor pendiente.
El módulo utilizará el límite establecido en la propiedad «Límite pendiente del cliente». Si esta propiedad no está configurada, puede definir un límite predeterminado para todos los clientes y un límite independiente para todos los proveedores.
