Installation
veuillez contacter si vous avez besoin d'une version pour http:// APK.
Si vos appareils ne prennent pas en charge l'application google play, veuillez contacter pour le téléchargement de l'apk.
Allez dans le menu Accueil - Configuration - Modules - Déployer un module externe et soumettez le fichier zip du module.
Le module "ExtDirect" est visible dans la liste des modules et peut être activé et configuré.

Fonctionnalités optionnelles sont inclus pour un utilisateur, un code pour révéler les options sera envoyé après votre achat.

Fonctionnalités
Gestion facile du produits – commandes – tiers sur des appareilles petit écran.
Gestion par lecture des code-barres automatique, pas besoin des champs de recherche.
Différentes manières de scanner les codes à barres: Avec le camera pour des scans occasionnelle.
Avec une douchette bluetooth pour des scans régulièrement.
Vérifiez ici les appareils compatibles. Avec un terminaux de lecture de codes-barres intégré pour des scans intensifs.
Corriger stock.
Transfert stock entre des entrepôts.
Adapté à l'utilisation de nombreux entrepôts.
Compatible gestion lot/série.
Ajouter le code-barre.
Ajouter photo produit.
Rassembler les produits commandes clients dans l’entrepôt.
Créer des expéditions.
Rassembler les produits de l'ordre de fabrication dans l'entrepôt.
Éditer ou créer des clients avec leur conditions de vente.
Créer – éditer – valider de commandes.
Accorder remise global.
