Installation
Avant d'acheter, s'il vous plaît installer l'application gratuite Mobilid Android de Google Play et tester la connexion de démo par défaut. (entrez un texte de votre choix dans le champ d'identification et appuyez sur le bouton retour). L'application de Google Play pour Android 9+ nécessite une version certifiée https:// à votre dolibarr, veuillez contacter si vous avez besoin d'une version pour http:// APK. Si vos appareils ne prennent pas en charge l'application google play, veuillez contacter pour le téléchargement de l'apk.
Allez dans le menu Accueil - Configuration - Modules - Déployer un module externe et soumettez le fichier zip du module.
Le module "ExtDirect" est visible dans la liste des modules et peut être activé et configuré.

Fonctionnalités optionnelles 'Réception', 'Achat' et 'Sortie de Stock' sont inclus pour un utilisateur, un code pour révéler les options sera envoyé après votre achat.
