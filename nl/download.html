Mobilid
Mobilid is een software oplossing bedoeld als een generieke mobiele client-server toepassing. U kan een universeel uitbreidbaar pakket kopen welke enkele standaard modules bevat. Uitbreidbaar wil zeggen dat u extra add-on modules en op maat gemaakte modules kan aankopen. Om deze modules te activeren, hebt u een gebruikers licentie nodig. Deze code dient ingevoerd te worden op de client applicatie. De mobiele client applicatie kan gratis gedownload worden.

Mobilid Dolibarr ERP connector

Dolibarr module om Mobilid te verbinden met het Dolibarr Open Source ERP systeem. Gebruik het Dolibarr ERP systeem om uw klant en product administratie uit te voeren. Via deze module ontwikkeld door Z-Application, kan Mobilid u helpen om de productiviteit te verhogen van uw bedrijfsadministratie.
Om het Dolibarr Open Source project te ondersteunen, kan deze module aangekocht worden via de Dolibarr module winkel.
