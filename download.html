Mobilid
Mobilid is a software package which intends to be a generic mobile client-server solution. You buy a extendable server-package including some standard modules. Extendable means that you can buy add-on modules or custom made modules. To enable these modules, you will need a user license code. You need to provide this code to the client application. The mobile client can be downloaded for free.
